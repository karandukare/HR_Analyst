# HR_Analyst

**Overview**

This particular project concentrates on applying SQL to examine the makeup of employees based on the data set in the Human Resources sector.
The aim is to take raw HR data and turn it into useful business intelligence that can assist HR professionals and management with their decisions.

**Key Highlights**

- Examined actual HR data related to both employee and
- Applied SQL queries to respond to business-related questions
- Emphasized employee numbers and gender diversity
- Organized, neat, and optimized SQL query scripts
- HR Analytics Project for Beginners

**Features**

- View and review employee records
- Find Total employees
- Analyze Employees by Gender
- Aggregating data to find workforce trends
- Provide insights relevant to HR decision-making

**Data Sources**

- HR Employee Data
- Data may include details about employees, including:
- Gender
- Department
- Employee files

**Business Problem and Analytics Problem**

*Business Problem*
The HR department must understand the following about the project:

- Employment level
- Gender ratio
- Diversity Balance within the Organization

*Analytics Problem:*
The raw data from the human resource function is difficult to interpret without any sort of analysis. The SQL language is employed to frame, summarize, and analyze data in order to answer various business questions related to the HR function.

**Key Insights from the Analysis**

- Total number of employees working in the organization
- Male-Female Ratio of Employees
- Transparent visibility into workforce composition
- Data informs diversity and hiring strategy choices

**The Solution Identified by the Code**

*The SQL queries:*

- Derive valuable insights from HR data
- Process raw employee data into key findings
- Assist in tracking diversity and workforce structure
- Support management in planning personnel recruitment and policymaking
